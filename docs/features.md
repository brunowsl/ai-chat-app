### Descrição do App - Aplicação Multitenant para Automação de Documentos Jurídicos

Imperativo: Este app deve ser utilizado por diferentes empresas contratantes onde cada empresa/cliente irá possuir seu espaço com automações customizadas pré criadas para cada um.

## Funcionalidades

1. Login/logout com keycloak
2. Aquisição de novos créditos [Admin]
3. Níveis de permissão para acesso a funcionalidades (Admin e Usuário default)
4. Visualização de uso de creditos por usuário e geral da sua empresa [Admin]
5. Execução de automações para geração de texto
6. Copiar para a area de transferência as respostas geradas pela IA
7. Visualização de uso de créditos do usuário
8. Area de billing com o status dos pagamentos e histórico de compra de créditos [Admin]
